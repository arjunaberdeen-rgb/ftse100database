# FTSE 100 Data Extraction Task

## Objective
The goal of this task is to extract historical price data for all companies currently listed in the FTSE 100 index.

## Specifications
- **Data Source**: Yahoo Finance (via `yfinance`).
- **Period**: 01-01-2020 to 01-01-2026.
- **Output**: A local storage of the extracted data (e.g., CSV or SQLite).
- **Script**: `database.py`

## Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Run the extraction: `python database.py`
